# PCM ESTRATÉGICO

Sistema SaaS multiempresa para gestão de manutenção industrial (PCM), com foco em controle operacional, segurança por empresa e produtividade das equipes.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Execução local

```sh
npm install
npm run dev
```
